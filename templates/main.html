[Click Me] To Show Modal Dialog

I think there are 2 options to do this. Have everything (apart from modal div) in a .content class, and then set content class opaque to .1 which nicely greys out everything and then show overlay (so user can't click content and then show message. Or....

Just show overlay(with opaque set) and message div - but this won't really grey everything out in the background.

If you don't know the height then you could just set the top, don't think it matters if it doesn't center vertically.
